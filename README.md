# Pokedex

This project is a **Pokémon Trainer Register (Pokedex)** built with **HTML, CSS and JavaScript**.

The application loads Pokémon data from an external **API** and displays them as interactive cards. Users can explore different Pokémon and view additional details in an expanded overlay.

## Features

* Fetch Pokémon data from an external API
* Render Pokémon cards dynamically
* Lazy loading for better performance
* Load more Pokémon with a button
* Detailed Pokémon view in a large overlay
* Smooth user feedback with loading indicators

## Technologies

* HTML
* CSS
* JavaScript
* REST API (PokéAPI)

## Project Goals

The goal of this project is to practice:

* Working with APIs
* Fetching and rendering dynamic data
* Implementing lazy loading
* Improving performance with caching
* Creating interactive UI components

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
